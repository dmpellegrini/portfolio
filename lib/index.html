Pokémern

As a long time fan of the Pokémon series growing up, I took the opportunity to utilize data from the robust Pokéapi restfull API. The API is a plethora of information but was overbuilt for the purposes of the app I was trying to create. So I cherry picked the data I needed from the disparate endpoints of the Pokéapi and seeded a custom Mongodb database. Then I built a custom API utilizing express to feed my front end app with the relevent data it needed. Finally with all the data in place, I utilized React in order to pull and display data from my custom Pokemon API. Click here to check out the code on github. And simply click on the project below to visit the website.
